# eymenkeyvan.com

Personal portfolio website for Eymen Keyvan — CS student, ML researcher, and software engineer.

## Tech Stack
- Vanilla HTML/CSS/JS with GSAP ScrollTrigger for animations
- Vite for build tooling
- Deployed on Vercel with custom domain via IONOS DNS

## Features
- Preloader animation with GSAP timeline
- Scroll-triggered reveal animations
- Responsive design (mobile-first)
- Warm cream color palette with editorial typography
- Contact form with mailto integration
- Infinite scrolling marquee

## Development
```bash
npm install
npm run dev
```

## Deploy
Push to `main` branch — Vercel auto-deploys.
